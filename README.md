# TOP Shop 

A fan-made merch store concept for the band [Twenty One Pilots](https://www.twentyonepilots.com/).
A single-page front-end application with a product catalog, real-time search, multi-filter system, and a shopping cart.

---

## Features

- Dynamic product card rendering from a JavaScript data array;
- Real-time search filtering by product name;
- Multi-select filter panel (by gender, size, category, and album era);
- Slide-in filter sidebar with overlay and `Escape` key support;
- Shopping cart with a live item counter;
- Discount badge on discounted products (auto-calculated);
- Album promotion banners with links to streaming platforms;
- Social media links section;
- Responsive design (mobile, tablet, desktop).

---

## Technologies

- HTML5;
- CSS3 (Flexbox, CSS Grid, CSS nesting, transitions, responsive design with media queries);
- JavaScript (DOM manipulation, dynamic rendering, array methods: `filter`, `forEach`, `includes`, `some`).

---

## How to Run

1. Download files or clone the repository:
   ```bash
   git clone https://github.com/druzdaniil/TOPshop.git
   ```
2. Open `index.html` in your browser — no build tools or server required.
